How to use:
First enter expression as in question, for example: x^2-2 (is expression for square root of 2). This algorithm is for finding roots. While entering the expression, variable should be 'x', power should be represented by '^', brackets should have symbol "*" to represent multiply and divide is "/" complex equation is like this: x*(x^2/2-1), and error should be written in decimal eg:0.0001. Itteration tab can be left empty for calculating in trems of only tolerance factor. But itterate a number of time write integer in itteration tab and random number after "0." in tolerance tab.
